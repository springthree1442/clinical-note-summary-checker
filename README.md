# AI 진료기록 요약 검토 시스템

이 프로젝트는 가상의 SOAP 형식 외래 진료기록을 바탕으로 생성형 AI의 일반 요약 방식과 구조화 요약 방식을 비교하는 Streamlit 웹앱입니다.

## 프로젝트 목적

병원 진료기록에는 증상, 병력, 알레르기, 검사 결과, 처방, 추후 계획 등 환자 안전과 관련된 정보가 포함됩니다. 생성형 AI는 긴 진료기록을 빠르게 요약할 수 있지만, 핵심정보를 누락하거나 원문에 없는 내용을 추가할 수 있습니다.

본 프로젝트는 일반 요약과 구조화 요약의 핵심정보 보존 정도를 비교하고, 의료진이 AI 요약문을 검토·수정·승인하는 보조 시스템의 형태를 구현하는 것을 목표로 합니다.

## 주요 기능

- 가상 SOAP 진료기록 사례 선택
- 원본 진료기록 확인
- 일반 요약문과 구조화 요약문 비교
- 핵심정보 보존 여부 체크
- 오류 유형 기록
- 최종 수정 요약문 작성
- 요약 방식별 정확도 표와 그래프 확인

## 사용 기술

- Python
- Streamlit
- Pandas
- Matplotlib
